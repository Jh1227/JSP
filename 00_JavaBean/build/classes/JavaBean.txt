# JavaBean
- 데이터를 표현하는 목적으로 작성하는 class 입니다
  > DB 데이터 처리
- 뷰페이지와 로직페이지를 서로 분리하기 위해서 만들어지는 class 입니다
- 하나의 테이블에서 DTO(VO), DAO 가 한 세트로 만들어 집니다

# DTO ( Data Transfer Object ), VO( Value Object )
- 데이터를 DB 로 보내거나, DB 로부터 레코드를 가져올 때 하나의 객체로 만드는 class 입니다

# DAO ( Data Access Object )
- 객체화된 DTO 를 가지고 DB 에 접속시 사용되는 메서드를 정의한 class 입니다

# JavaBean 규칙
- class 는 반드시 특정 package 소속이어야 합니다
- DTO 의 멤버필드를 property 라고 합니다
- property 의 접근지정자는 private 으로 설정해야 합니다
- 멤버필드마다 별도의 getter/setter 메서드가 있어야 합니다
- getter/setter 메서드의 접근지정자는 public 이어야 합니다
- 매개변수가 없는 기본생성자가 있어야 합니다
